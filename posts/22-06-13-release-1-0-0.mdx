**Seven23 will now follow the [Semantic Versioning](https://semver.org/) specifications** to provide more visibility regarding its release cycle. **On June 8th, 2022, a first version has been released** on both client and server side.

## Client

The **main interface** is design as a progressive web application. Source code is **publicly available** within the [seven23](https://github.com/sebastienbarbier/seven23) repository. All changes will be logged in the [CHANGELOG.md](https://github.com/sebastienbarbier/seven23/blob/main/CHANGELOG.md) file at the repository root level.

**First release** provide the following description:

```
## [1.0.0] - 2022-06-08
### âœ¨ Feature
- List transactions, categories, and exchange rates.
- Offline enabled.
- Recursive transactions.
- Display your transactions using any exchanged currency.
- Multi device sync with end-to-end encryption.
- Responsive design for mobile, tablet, and desktop.
- Dark theme.
- Multiple accounts.
- Dashboard view with trends and summaries.
- Report generator for selected date range.
- Search engine for your transactions.
- Convertor based on provided rates.
- Import/Export your data as JSON.
- Hide numeric values.
- Nomadlist integration.

```
